# Mithaas Junction 🍬

Mithaas Junction is a simple and responsive sweet recipe website built using HTML and CSS.  
The website showcases popular Indian sweets with ingredients, recipes, and attractive hover effects.

---

## ✨ Features

- Beautiful navigation bar
- Sweet recipe cards
- Hover overlay effect on images
- Clean and responsive layout
- Modern UI with Flexbox
- Gradient backgrounds and smooth hover animations

---

## 🍭 Recipes Included

- Gulab Jamun
- Barfi
- Kalakand
- Rasmalai

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
